# Codename Eagle server list

Gamespy master server replacement that only accounts for Codename Eagle.

## Running

Requires Node.js 26 or newer (TypeScript runs directly via type-stripping, no build step).

```sh
git clone git@github.com:rexxars/ce-server-list.git
cd ce-server-list
npm install
npm start
```

For development with auto-reload and pretty logs:

```sh
npm run dev
```
